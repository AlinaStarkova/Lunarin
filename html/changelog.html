  • Added a new alert for n2o in the air

  • Clowns are once again the only beings capable of controlling the clown car, it should be fully functional again
  • Infectious zombies (from romerol) can actually be wounded and dismembered (and thus decapitated) properly

Seris02 updated:

  • makes it so that chat logs respect message types

TemporalOroboros updated:

  • Modular computers can log things without spamming runtimes again.
  • Fixed simple animal mecha pilots boarding mechas.
  • The toxin purge nanite programs no longer runtime when applied to simple mobs.

cacogen updated:

  • The tape recorder makes sounds when you interact with it

coiax updated:

  • Hydroponics trays on examine now print messages about weeds and pests in the right order.
  • Examining a hydroponics tray by clicking it when adjacent as a human now gives the same message as when examining.

prodirus updated:

  • Tape recorder cassette can no longer be rewound with a pen if it is not unwound.
  • Tape recorder transcript should look like it has text written on it!

  • Food/slime processor should no longer be able to turned on while empty.
  • Doorjacks should be able to hack green airlocks now /emags should not be able to hack green airlocks now

ArcaneDefence updated:

  • Katanas when worn on the belt slot are consistently worn on the left hip.

Arkatos updated:

  • Scrolls of teleportation no longer use outdated interface and will take you straight into a list of a possible destinations upon use.
  • Scrolls of teleportation now show number of remaining uses upon examination and also give proper feedback messages upon teleportation.
  • Scrolls of teleportation now crumble to dust when out of uses.

Azarak updated:

  • fixed the recycler eating all sorts of effects, including directional lighting

MissFox0810 updated:

  • Added playtime to messages SQL table

  • Griddles no longer eat crowbars
  • Icebox cavegen generates the correct mobs and ore on the bottom z-level again
